# 🏀 NBA Milestones - Quick Update Guide

## Easiest Way to Update Stats

### Windows (Double-click)
📁 **Double-click `UPDATE_STATS.bat`** in your project folder

### OR use terminal command
```bash
npm run update
```

That's it! Both methods do the same thing:
1. ✅ Fetch latest NBA games (~30-40 min)
2. ✅ Rebuild leaderboards (~3-5 min)  
3. ✅ Update production website automatically

## When to Update
- 📅 **Daily** during NBA season
- 📅 **Weekly** if you check less often
- 🎯 **After big games** for your watchlist players

## Troubleshooting

**Script hangs?** → Let it run, NBA API is slow  
**Stats not showing?** → Click "Refresh milestones" on watchlist  
**Any errors?** → See `docs/MANUAL_UPDATE.md` for detailed help
